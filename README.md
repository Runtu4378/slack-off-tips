# slack-off-tips

摸鱼办公室文案生成器

## 示例

```
【摸鱼办公室】9月13日
上午好，摸鱼人，工作再累，一定不要忘记摸鱼哦！
有事没事起身去茶水间去厕所去廊道走走，别老在工位上坐着，钱是老板的，但命是自己的
距离本周周末还有5天
距离中秋假期还有5天
距离国庆假期还有17天
距离跨年假期还有109天
距离春节假期还有140天
```

## 指令

```bash
# 安装
npm i -g https://github.com/Runtu4378/slack-off-tips

# 显示今天的摸鱼文案
moyu show
# 增加自定义节日锚点
moyu mark 2021-12-25 圣诞节`
```
